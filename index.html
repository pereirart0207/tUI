What is tUI?

tUI is a tiny JavaScript module that lets you display stylish spinners, toast notifications, and modals without relying on any frameworks or external libraries.

Key Features

  • ✅ Ultra lightweight - Only 6.1 KB minified
  • ✅ Zero dependencies - Works with any framework or vanilla JS
  • ✅ Easy to use - Simple API with sensible defaults
  • ✅ Customizable - Colors, sizes, durations and more
  • ✅ Accessible - Proper ARIA attributes and keyboard support
  • ✅ Modern design - Clean, professional UI components

Quick Start

1. Include the script

Add tUI to your project with a simple script tag:

<script src="tui.js"></script>

2. Start using the components

Basic usage examples:

// Show a loading spinner
tUI.showSpinner({ message: "Loading..." });

// Show a success notification
tUI.showNotification({ 
  message: "Operation completed!", 
  type: "success" 
});

// Show a modal with custom HTML
tUI.showModal("<h2>Welcome</h2><p>This is a custom modal.</p>");

// Hide the spinner when done
tUI.hideSpinner();

// Close all modals
tUI.closeModal();

Detailed Documentation

Spinner Component

Show loading indicators during async operations.

Methods

tUI.showSpinner(options);
tUI.hideSpinner();

Options

Option Type Default Description
message String "Loading..." Text displayed below the spinner
color String "#333" Color of the spinner animation
size Number 50 Diameter of the spinner in pixels
background String "#fff" Background color of the spinner container

Toast Notifications

Display temporary messages to users.

Methods

tUI.showNotification(options);

Options

Option Type Default Description
message String "Notification" The text content to display
type String "info" One of: "success", "error", "warning", "info", "dark"
duration Number 4000 Time in milliseconds before auto-dismiss
backgroundColor String null Override the default type color
color String "#fff" Text color
fontSize String "14px" Font size of the message

Example

tUI.showNotification({
  message: "File uploaded successfully",
  type: "success",
  duration: 5000
});

Modal Dialogs

Display custom content in centered dialogs.

Methods

tUI.showModal(htmlContent);
tUI.closeModal();

Parameters

Parameter Type Description
htmlContent String HTML string to display in the modal

Example

tUI.showModal(`
  <h2>Confirm Action</h2>
  <p>Are you sure you want to delete this item?</p>
  <div style="display: flex; gap: 10px; margin-top: 20px;">
    <button onclick="deleteItem(); tUI.closeModal()" 
            style="padding: 8px 16px; background: #f44336; color: white;">
      Delete
    </button>
    <button onclick="tUI.closeModal()" 
            style="padding: 8px 16px; background: #ccc;">
      Cancel
    </button>
  </div>
`);
