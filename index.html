What is tUI?

tUI is a minimalist JavaScript library that provides essential UI components for web applications without the bloat of larger frameworks. It's perfect when you need just the basics done right.

Why Choose tUI?

  • Ultra lightweight - Only 6.1 KB minified (2.8 KB gzipped)
  • Zero dependencies - Works with any stack or vanilla JS
  • Simple API - Intuitive methods with sensible defaults
  • Fully customizable - Colors, sizes, durations and more
  • Accessible - Proper ARIA attributes and keyboard navigation
  • Modern design - Clean, professional UI out of the box
  • MIT Licensed - Free for personal and commercial use

Quick Start

Current version: 1.0.0 ¬∑ License: MIT ¬∑ Compatibility: All modern browsers

1. Installation

Add tUI to your project with a simple script tag:

<script src="https://cdn.jsdelivr.net/npm/tui@1.0.0/dist/tui.min.js"></script>

Or install via NPM:

npm install tui-module

2. Basic Usage

Here's how to use the core components:

// Show a loading spinner
tUI.showSpinner({ 
  message: "Processing your request...",
  color: "#4CAF50"
});

// Show a success notification
tUI.showNotification({
  message: "Settings saved successfully!",
  type: "success",
  duration: 5000
});

// Show a custom modal dialog
tUI.showModal(`
  <h2>Confirm Action</h2>
  <p>Are you sure you want to proceed?</p>
  <button onclick="tUI.closeModal()">Cancel</button>
  <button onclick="confirmAction()">Confirm</button>
`);

// Hide components when done
tUI.hideSpinner();
tUI.closeModal();

Documentation

Spinner Component

Display loading indicators during asynchronous operations.

Methods

tUI.showSpinner(options);  // Shows loading spinner
tUI.hideSpinner();         // Hides the spinner

Options

Option Type Default Description
message String "Loading..." Text displayed below the spinner
color String "#333" Color of the spinner animation
size Number 50 Diameter of the spinner in pixels
background String "rgba(255,255,255,0.9)" Background color of the spinner container
zIndex Number 9998 Z-index of the spinner container

Toast Notifications

Show temporary messages to users with different status types.

Methods

tUI.showNotification(options);

Options

Option Type Default Description
message String "Notification" The text content to display
type String "info" Type: "success", "error", "warning", "info", "dark"
duration Number 4000 Time in milliseconds before auto-dismiss
position String "top-right" Position: "top-right", "top-left", "bottom-right", "bottom-left"
backgroundColor String null Override the default type color
color String "#fff" Text color
fontSize String "14px" Font size of the message

Example

tUI.showNotification({
  message: "Profile updated successfully",
  type: "success",
  duration: 5000,
  position: "top-right"
});

Modal Dialogs

Display custom content in centered, focus-trapped dialogs.

Methods

tUI.showModal(content, options);  // Shows modal with HTML content
tUI.closeModal();                // Closes all open modals

Options

Option Type Default Description
width String "600px" Width of the modal content
background String "rgba(0,0,0,0.5)" Overlay background
closeOnClickOutside Boolean true Close modal when clicking outside
closeOnEsc Boolean true Close modal when pressing ESC key
zIndex Number 9999 Z-index of the modal

Advanced Example

tUI.showModal(`
  <div style="padding: 20px;">
    <h2 style="margin-top: 0;">Advanced Modal</h2>
    <p>This modal has custom buttons and styling.</p>
    <div style="display: flex; gap: 10px; margin-top: 20px;">
      <button onclick="tUI.closeModal()" 
              style="padding: 10px 20px; background: #ccc; border: none; border-radius: 4px;">
        Cancel
      </button>
      <button onclick="submitForm(); tUI.closeModal()" 
              style="padding: 10px 20px; background: #4CAF50; color: white; border: none; border-radius: 4px;">
        Submit
      </button>
    </div>
  </div>
`, {
  width: "500px",
  closeOnClickOutside: false
});

Frequently Asked Questions

Can I use tUI with React/Vue/Angular?

Yes! tUI is framework-agnostic and works well with any JavaScript framework or vanilla JS.

How do I customize the look and feel?

All components accept style options through their configuration objects. You can also override the CSS classes for deeper customization.

Is tUI accessible?

Yes, all components include proper ARIA attributes and keyboard navigation support.

Can I animate the components?

Absolutely. The toasts and modals include simple fade animations, which you can customize or disable via CSS.
